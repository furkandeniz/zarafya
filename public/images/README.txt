Zarafya logo paket içeriği:
- zarafya_logo_master_transparent.png : Web için ana logo (şeffaf zemin).
- zarafya_logo_black.png / zarafya_logo_white.png : Tek renk (monokrom) PNG (şeffaf zemin).
- zarafya_logo_grayscale.png : Gri ton.
- zarafya_logo_black_on_white.png / zarafya_logo_white_on_dark.png : Örnek arka planlı versiyonlar.
- zarafya_icon_*.png : Kare ikon seti (16–1024).
- favicon.ico : 16/32/48 çoklu boyut favicon.
- *embedded.svg dosyaları*: PNG’yi SVG içine gömülü şekilde içerir (vektör path değildir).

Not: Gerçek vektör (path’li) SVG için otomatik vektörleştirme aracı (potrace vb.) gerekir; bu ortamda yok.
Eğer istersen, bir “vektörleştirme yönergesi” ve önerilen ayarlarla (Illustrator/Inkscape) 2-3 dakikada temiz SVG elde edebilirsin.
